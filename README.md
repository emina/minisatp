MiniSat-p v1.14
========================================

This is a patched and slightly extended version of the MiniSat v1.14 solver 
with proof logging.  The original source code, by Niklas Een and 
Niklas Sörensson, is [available](http://minisat.se/downloads/MiniSat-p_v1.14.2006-Sep-07.src.zip) 
at the [MiniSat page](www.cs.chalmers.se/Cs/Research/FormalMethods/MiniSat/).
